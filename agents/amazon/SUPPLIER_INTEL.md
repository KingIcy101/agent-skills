## ⚠️ Critical Rules

- **ONCE A MONTH per brand per account** — ordering more frequently raises flags. Space orders ~4 weeks apart minimum.
- **VARY ORDER SIZES** — don't order the same quantity every time. Variation is intentional.
- **Multi-account is intentional** — split volume across accounts to stay under brand radar
- **BodyHealth flagging NOW** — keep per-account orders small and spaced out
- **Accounts get shut down** — never route to accounts without confirming they're still active
- **Rebecca has markup** — charges more per unit but allows larger Ortho + BH orders
- **DUFFLE = from Mateo** — Mateo is the supplier; Matt orders through DUFFLE for extra capacity. "Matt X" tabs = Matt's actual orders placed through those contacts.
- **Momentous = low profit, skip** — don't plan restock orders unless Matt explicitly asks
- **Sellerboard = source of truth** for what sells and what's profitable
- **Study order history before flagging any restock** — understand each account's cadence

## Kargo Restock Logic (v2)

```
For each brand/product needing restock:

1. Pull velocity + current stock from Sellerboard
2. If days_remaining < 14 → flag for reorder
3. Check when each account last ordered for this brand
   → Must be ≥ 4 weeks since last order at that account
4. Route to account with longest time since last order
5. VARY the quantity from last time (intentional)
6. Spread across multiple accounts if volume is large
7. Never double-up at same account less than 4 weeks apart
8. Flag for Matt review before placing
9. Never auto-place — flag only
```

**Per-brand routing priority:**
- **BodyHealth:** Carol/Angelica first → Mike/Suzanne/Renee as needed → Rebecca overflow (markup) → DUFFLE last resort
- **Ortho:** Rebecca (larger orders ok, markup) / Renee / Suzanne → Mike (10% at 144+) → Marco (confirm active)
